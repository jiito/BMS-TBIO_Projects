Introduction

nibls is a program for the Affymetrix Eureka™ platform which works on the given sequence data and generates output files that can then be used by the viewer appliation to display data analytics

Quick Start

Binning can be performed from the command line as follows: 

  nibls.exe --config <file>

Options 

  -h, --help            show this help message and exit
  -c <file>, --config <file>
                        path to configuration file

For more information about JSON configuration format see Eureka Analysis Suite User Manual.
