Introduction

apt-copynumber-format-igv is an application which converts probe level and segment level data into a format compatible with the Integrative Genome Viewer (IGV).

Quick Start

To export probe level/segment data into IGV format, use:

: 

    apt-copynumber-format-igv [options] file1.chp ... fileN.chp

Options:

Usage: 

exports the Copy Number Segment and Log 2 Ratio data from the list of chp files in MYCHPFILES.txt.  The exported files will have the prefix names myfilename.cn.seg and myfilename.log2ratio.cn

apt-copynumber-format-igv –out-dir IGVEXPORT –-sample-list-file MYCHPFILES.txt –export-segment-cn –export-cn-log2ratio –output-prefix myfilename

--sample-file-list: text file containing the list of chp file names.  lnclude the entire path where files reside. No headers

--out-dir: Name of the output directory where igv exports will  be saved

--export-seg-loh: Exports all LOH segments from the chp file(s). 

--export-seg-cn: Exports all Copy Number segments from the chp file(s) 

--export-cn-log2ratio: Exports the log 2 ratio probe level data from the chp file(s)

--export-cn-weighted-log2ratio: Exports the weighted log 2 ratio probe level data from the chp file(s)

--export-cn-allele-difference: Exports the Allele Difference probe level data from the chp file(s)

--export-allele-baf: Exports the B-Allele Frequency (BAF) probe level data from the chp file(s)

--export-cn-smooth-signal: Exports the Smooth Signal probe level data from the chp file(s)

--export-sample-attributes: These files are generated by the ChAS GUI. This command will copy the _sampleattributes.txt file residing with the chp file to the designated --out-dir

--output-prefix: names the output file with a designated prefix. else exported files are named using date_time.suffix
