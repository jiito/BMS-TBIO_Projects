CONTENTS 
========

LINUX/OS X (BINARY DISTRIBUTION):
---------------------------------
- bin - Affymetrix Power Tools (APT) executable programs.

- doc/dox-html - Documentation, help, vignettes and release notes.


WINDOWS:
--------
- C:\Program Files\Thermo Fisher Scientific\Affymetrix Power Tools\apt-1.20.5\bin - Affymetrix Power Tools (APT) executable programs for windows

- C:\Program Files\Thermo Fisher Scientific\Affymetrix Power Tools\apt-1.20.5\doc - Documentation, help, vignettes and release notes



INSTALLATION FROM BINARY DISTRIBUTIONS
======================================

LINUX/OS X: 
-----------
If installing from a binary distribution, copy the files to a dir on your $PATH or add the bin directory to your path:

   export PATH=apt-1.20.5/bin:$PATH

WINDOWS 64-bit: 
---------------
unzip APT-1.20.5-i686-intel-win64.zip, click APT-1.20.5-i686-intel-win64.msi file and follow the instructions to install windows 64-bit.


NOTE FOR WINDOWS USERS: These executables should be run from an account with Administrator privileges on your system.
