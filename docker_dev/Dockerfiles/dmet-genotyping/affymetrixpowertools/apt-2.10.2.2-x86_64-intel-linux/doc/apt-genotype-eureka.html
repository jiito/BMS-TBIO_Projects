Introduction

apt-genotype-eureka is a program for making genotyping calls on the Affymetrix Eureka™ platform. apt-genotype-eureka uses the BRLMM-P algorithm for genotype calling. BRLMM-P is a model-based approach which performs 1-dimensional clustering by fitting a Gaussian mixture model.

Quick Start

The basic requirements for a run of apt-genotype-eureka are

  • a binning file generated by the nibls application;
  • a priors file.

Additional parameters may be specified on the command line or by means of a parameter XML file.

The command below runs the basic genotyping analysis on a Eureka binning file:

  apt-genotype-eureka \
      --chip-type eureka \
      --binning-file ./input/Binning.txt \
      --snp-priors-input-file ./inputs/referencePriors_heavy_CES.txt \
      --out-dir out \
      --cel-files cel_file_list.txt \
      --log-file ./apt-genotype-eureka.log
