Introduction

apt-copynumber-cnvmix.exe is a program that reports copy number in predefined regions with dynamic log2ratio calling thresholds. The program also can report on regions lacking heterozygosity (LOH).  Both features require a cn_models library file with necessary information.”

Quick Start

Command-line Parameters

apt-copynumber-axiom-cnvmix 

--out-dir .\out_dir

--mapd-max 0.35

--waviness-sd-max 0.1

--chip-type .\Axiom

--use-text-format-summary-file False

--summary-file .\AxiomGT1.summary.a5

--report-file .\AxiomGT1.report.txt

--arg-file .AxiomCNVmix.apt2.xml

--analysis-files-path \libraryfilepath

--cn-priors-file .cn_priors

--reference-file .cn_models

--log-file .\logfile.out
